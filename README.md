# zima-fip

# Zima Project
This is the Zima project, a collaborative effort between Wisdom Okutepa and Osarieme Ogbeide. Zima is a fictional brand focused on crafting fine beverages with simplicity and quality.

# Project Overview
Zima is a responsive web project aimed at showcasing the brand's products and providing information about the company and its offerings.

# Technologies Used
## HTML
## CSS (including CSS Grid and Flexbox for layout)
## JavaScript (for interactive elements like lightbox)

# Project Structure

The project directory structure is as follows:

### index.html: Main landing page for the website.
### products.html: Page showcasing Zima's product selection.
### contact.html: Contact page for reaching out to the company.
### css/: Contains CSS files for styling.
### main.css: Main styles for the website.
### grid.css: CSS Grid styles for layout.
### images/: Directory for storing images used in the project.
### video/: Directory for storing videos used in the project.
### js/: Contains JavaScript files for interactive features.
### README.md: Documentation file (what you're currently reading).

# How to Run To run the project locally:
Clone the repository to your local machine.
Open the project directory in your code editor.
Open the index.html file in your web browser to view the main landing page.
